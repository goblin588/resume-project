Work Experience

Griffith University Centre for Quantum Dynamics

Jan 23 - Present

Participated in a seven-week experimental research internship at the Quantum Optics Lab. The role involved hands-on experience in lab optics and the application of machine learning to optimize 'beam walking' coupling.

Had some lab experience building a 'beam walking' coupling setup and manually coupling with single mode fibres then over the course of the 7 weeks. developed an algorithm using Gaussian estimation methods to predict and iteratively enhance the alignment of 4 axial parameters (two Zaber TMM-2 motor-mounted mirrors) based off the output measured with a power meter in order to effectively determine the most optimal mirror positions for alignment.

The project was coded in Python, though Git and GitHub were also used for version control systems in VSCode, enabling regular progress tracking and software accessibility for other researchers.

The outcome of this internship was a working model which could successfully optimise the set up though there was more work to be completed in cleaning up the code, making it more readable and turning it into a library as well as work to improve efficiency of optimisation and including some methods for displaying the data aquired during runs for analysis.

I really enjoyed this experience and significantly developed my skills in programming, and problem-solving amongst having the chance to talk to many of the amazing researchers in CQD.

I was invited to continue working on the project which i started summer 2023. The project was about development of an algorithm to optimize laser to single mode fiber coupling. This extension of the project would see the algorithm expanded from 2 to 4 axial dimensions, and wrapping the scripting up and migrating onto GitHub with other features so that the program can be installed and used by researchers in the lab.

The github repositiory i used for this project is available here with a README doc that explains the project in further detail.
